# Instalação ![test](https://icon-icons.com/icons2/54/PNG/64/installation_application_software_10808.png)
* Clone o Repositorio
` 
    $ git clone https://github.com/RavokTHC/THC-Word-List.git
`

* Descompacte o Arquivo
`
    $ unzip HC-Word-List-master.zip
 `  
* Conceda as Permissoes
`
    $ chmod +x THCList.py
 `   
* Execute o Script
`
    $ python THCList.py
 `
